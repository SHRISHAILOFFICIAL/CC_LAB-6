# CC Lab 6 – Jenkins & NGINX Load Balancing

This project demonstrates a CI/CD pipeline using Jenkins, Docker, and NGINX.

## Features

- Backend application containerized using Docker
- Automated build and deployment using Jenkins Pipeline
- Multiple backend containers deployed dynamically
- NGINX configured as a load balancer
- Supports Round Robin, Least Connections, and IP Hash strategies

## Technologies Used

- Jenkins
- Docker
- NGINX
- GitHub

This lab showcases modern DevOps practices including containerization,
automation, and load balancing.
